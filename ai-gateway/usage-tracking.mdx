View live metrics:

- **Success rate**: Percentage of successful requests
- **Average latency**: Mean response time
- **Cost per hour**: Current spending rate

### Budget Alerts

Set spending limits:

```typescript
await regpilot.usage.setBudget({
  amount: 1000,
  period: "monthly",
  alert_thresholds: [50, 75, 90, 100],
  actions: {
    at_90: "notify_admins",
    at_100: "disable_keys"
  }
});
```

## Usage Analytics API

Access usage data programmatically:

```typescript
// Get usage for date range
const usage = await regpilot.usage.get({
  startDate: '2024-01-01',
  endDate: '2024-01-31',
  groupBy: 'day'
});

// Get costs breakdown
const costs = await regpilot.usage.costs({
  currency: 'USD',
  groupBy: 'model'
});

// Get performance metrics
const performance = await regpilot.usage.performance({
  metric: 'latency',
  aggregation: 'p95'
});
```
